# 有限注德州扑克（Limit Texas Hold'em）完整规则

---

## 一、概述

有限注德州扑克（Fixed-Limit / Limit Texas Hold'em）是德州扑克的经典玩法之一。与无限注不同，**每次下注和加注的金额是固定的**，玩家不能随意下注任意金额。

---

## 二、盲注结构

以一个 **$2/$4** 的有限注牌局为例：

| 术语 | 金额 | 说明 |
|------|------|------|
| **小盲注（SB）** | $1 | 庄家左边第一位，强制下注 |
| **大盲注（BB）** | $2 | 庄家左边第二位，强制下注 |
| **小注（Small Bet）** | $2 | 前两轮的固定下注单位 |
| **大注（Big Bet）** | $4 | 后两轮的固定下注单位 |

> 📌 牌局命名规则：**$2/$4** 中，$2 是小注，$4 是大注。小盲注通常为小注的一半（$1），大盲注等于小注（$2）。

---

## 三、四轮下注详细规则

### 🃏 第一轮：翻牌前（Preflop）

- **发牌**：每人发 2 张底牌（暗牌）
- **下注单位**：**小注（$2）**
- **起始行动者**：大盲注左边第一位（UTG，枪口位）
- **可选动作**：

| 动作 | 说明 |
|------|------|
| **弃牌（Fold）** | 放弃本局 |
| **跟注（Call）** | 投入 $2（与大盲注持平） |
| **加注（Raise）** | 投入 $4（即 $2 跟注 + $2 加注） |

- 如有人加注到 $4，后续玩家可以：
  - 弃牌
  - 跟注 $4
  - **再加注到 $6**（再加一个小注 $2）

> ⚠️ **每轮最多允许 1 次下注 + 3 次加注**（即封顶为 $8），部分赌场规则为 1 次下注 + 4 次加注。**只剩两人单挑（Heads-up）时，通常取消加注次数限制。**

```
翻牌前加注阶梯（$2/$4牌局）：
  下注: $2 → 加注: $4 → 再加注: $6 → 封顶(Cap): $8
  每次增加一个小注 $2
```

---

### 🃏 第二轮：翻牌圈（Flop）

- **发牌**：翻出 3 张公共牌
- **下注单位**：**小注（$2）**
- **起始行动者**：庄家左边第一个未弃牌的玩家
- **可选动作**：

| 动作 | 说明 |
|------|------|
| **过牌（Check）** | 本轮还没人下注时，选择不下注 |
| **下注（Bet）** | 投入 $2 |
| **跟注（Call）** | 跟前面的下注金额 |
| **加注（Raise）** | 在下注基础上再加 $2 |
| **弃牌（Fold）** | 放弃本局 |

```
翻牌圈加注阶梯（$2/$4牌局）：
  下注: $2 → 加注: $4 → 再加注: $6 → 封顶(Cap): $8
```

---

### 🃏 第三轮：转牌圈（Turn）

- **发牌**：翻出第 4 张公共牌
- **下注单位**：⬆️ 提升为 **大注（$4）**
- **起始行动者**：同翻牌圈
- **规则同上，但金额翻倍**：

```
转牌圈加注阶梯（$2/$4牌局）：
  下注: $4 → 加注: $8 → 再加注: $12 → 封顶(Cap): $16
  每次增加一个大注 $4
```

---

### 🃏 第四轮：河牌圈（River）

- **发牌**：翻出第 5 张公共牌（最后一张）
- **下注单位**：**大注（$4）**
- **规则与转牌圈完全一致**

```
河牌圈加注阶梯（$2/$4牌局）：
  下注: $4 → 加注: $8 → 再加注: $12 → 封顶(Cap): $16
```

---

## 四、每轮结束的条件

> **当所有未弃牌的玩家都已行动，且投入的筹码金额相同时，本轮结束。**

### 具体场景：

| 场景 | 本轮是否结束？ |
|------|-------------|
| 所有人都过牌（Check） | ✅ 结束 |
| 有人下注，所有人跟注或弃牌，没人再加注 | ✅ 结束 |
| 有人加注，所有人跟注或弃牌，没人再加注 | ✅ 结束 |
| 加注次数达到封顶（Cap），所有人跟注或弃牌 | ✅ 结束 |
| 有人加注，还有人未行动或未补齐筹码 | ❌ 未结束 |

### 翻牌前的特殊情况：
- 如果前面所有人只是跟注（没有加注），**大盲注还有一次行动机会**，可以选择：
  - **过牌（Check）**：满意当前下注额，本轮结束
  - **加注（Raise）**：此时其他玩家需要再次回应

## 五、完整示例（$2/$4 有限注，6人桌）

```
位置：  庄家(D) → 小盲(SB) → 大盲(BB) → UTG → MP → CO

【翻牌前】（小注 $2）
SB: 强制下注 $1
BB: 强制下注 $2
UTG: 跟注 $2
MP:  加注到 $4          ← 第1次加注
CO:  再加注到 $6         ← 第2次加注
D:   弃牌
SB:  弃牌
BB:  跟注 $6（补 $4）
UTG: 跟注 $6（补 $4）
MP:  跟注 $6（补 $2）
→ 所有人 $6，本轮结束 ✅ 底池 = $1(SB弃牌损失) + $6×4 = $25

【翻牌】发出 3 张公共牌（小注 $2）
BB:  过牌
UTG: 下注 $2             ← 第1次下注
MP:  跟注 $2
CO:  加注到 $4            ← 第1次加注
BB:  弃牌
UTG: 再加注到 $6          ← 第2次加注
MP:  跟注 $6（补 $4）
CO:  跟注 $6（补 $2）
→ 所有人 $6，本轮结束 ✅

【转牌】发出第 4 张公共牌（大注 $4）
UTG: 下注 $4
MP:  跟注 $4
CO:  跟注 $4
→ 所有人 $4，本轮结束 ✅

【河牌】发出第 5 张公共牌（大注 $4）
UTG: 下注 $4
MP:  弃牌
CO:  跟注 $4
→ 所有人 $4，本轮结束 ✅

【摊牌】UTG 和 CO 比较手牌大小，大者赢得底池
```

## 六、有限注 vs 无限注 对比

| 项目 | 有限注（Limit） | 无限注（No-Limit） |
|------|---------------|-------------------|
| 每次下注/加注金额 | **固定**（小注或大注） | **任意金额**（最低有要求） |
| 最大单次下注 | 封顶金额（Cap） | 全部筹码（All-in） |
| 每轮加注次数 | **限制 3-4 次** | **无限制** |
| 前两轮下注单位 | 小注 | 无固定，最低为大盲注 |
| 后两轮下注单位 | 大注（小注的2倍） | 无固定，最低为大盲注 |
| 波动性 | 较小，风险可控 | 较大，一手可能全输 |
| 策略侧重 | 价值下注、精确计算赔率 | 诈唬、下注量控制、心理博弈 |

## 七、关键规则速记

```
📌 前两轮（翻牌前 + 翻牌）→ 用小注
📌 后两轮（转牌 + 河牌）  → 用大注（= 小注 × 2）
📌 每轮最多加注 3-4 次（到Cap封顶）
📌 单挑时通常无加注次数限制
📌 大盲注在翻牌前有最后行动权（Option）
📌 每轮结束条件：所有人行动完毕 + 金额一致
